# Jira SLA Excel Processor (Frontend)

Frontend for the Jira SLA calculating tool based on the given metrics.
Upload an Excel file containing Jira SLA data, and download a processed version generated by the backend.

## Workflow

1. User uploads Excel file via UI
2. File is sent to backend API
3. Backend processes Jira SLA data
4. Backend returns updated Excel file
5. User downloads result

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Setup

### Prerequisites

- Node.js 16+
- Backend API running (see backend repo)

### Environment Variables

Create a `.env` file:
VITE_API_URL=http://localhost:3000

### Run

npm install
npm run dev

Open http://localhost:5173
