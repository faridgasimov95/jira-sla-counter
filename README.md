# Jira SLA Counter

A web tool for processing Jira ticket SLA data. Fetches ticket status history from Jira and calculates total time spent in "Second line" support.

## Structure

- [Frontend](./frontend/README.md)
- [Backend](./backend/README.md)

## Setup

1. Set up the backend first — see [backend README](./backend/README.md)
2. Set up the frontend — see [frontend README](./frontend/README.md)
3. Start both servers, then open http://localhost:5173 in your browser

## Requirements

- Node.js 16+
- Jira account with an API token
